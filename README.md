# ACCELA_STEAMDECK
Downloader and install games for Steam OS

Esté é um Fork do Projeto ACCELA.

O código original está mantido, não sou o criador do projeto.

Foi acrescentando opção de selecionar a api de download. pública ou morrenus.

Também foi adicionado a opção de modificar o tamanho dos botões para ter uma melhor experiência de uso no modo game do Steam Deck.

Os creditos ao projeto vai aos criadores anonimos.
